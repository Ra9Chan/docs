How to code #

  1. clear goal at the beginning (清晰的目标)
  2. blindly copy the code without caring how it word (盲目复制代码,不关心如何运行)
  3. didn't reinforce (没有强化技能和做笔记)
  4. didn't practice coding (在学习教程外通过编程提高)
  5. quickly became frustrated (很快会失去兴趣)
  6. lost motivation (无法掌握全部内容)
  7. learning in isolation (独狼式学习 没有分享 无法获得满足)

fix P52 #

  1. set specific,measurable, realistic and time-based goal(设置一个具体, 可衡量, 现实, 基于时间的目标)
  2. why learn code: Switching careers(帮助保持动力在遇到困难的时候)
  3. imagine a big project (想象一个大项目,学习需要的技术) ----------Question2---------
  4. Understand the code when learning(学习的时候要确保理解再继续)
  5. Alway type the code ,don't copy-paste!(输入代码 不要复制)
  6. learn new feature or concep use it immediately(学习新的特性和概念,立即使用它强化记忆和理解)
  7. take notes and challenge self(做笔记和发现一些强化的小挑战关于现实生活) ----------Question3---------
  8. leave tutorial hell (远离教程陷阱) ----------Question4---------
  9. Don't get stuck trying to write the perfect code(不要因为写不出好的代码而失望)
  10. you will never you know everything(不断学习,没有人生来就是专家)
  11. focus your goal (专注目标) ----------Question5---------
  12. 分享新学内容,解释它,可以成功解释说明了解
  13. share goal to make yourself accountable(通过分享目标计划激励自己)
  14. 分享可以获得反馈和其他人的激励 ----------Question6---------
  15. Courses is a starting point
  16. Keep challenging yourself

4 STEPS to solve any problem #

  1. 100% understand the problem,Ask right questions to get a clear picture(100%理解问题,问正确的问题了解需求)
  2. Divid and conque(把大问题分解为小问题,直到完全了解、理解) look like task list(分而治之)
  3. don't be afraid to da as much research(使用 google，stackoverflow，MDN 搜索解决问题)
  4. write pseudo-code before writing the actual code(通过伪代码理清思路)
